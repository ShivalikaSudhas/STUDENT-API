# Student Management API

## Description
REST API built using Node.js HTTP module (without Express).

## Features
- GET all students
- GET student by ID
- POST create student
- PUT update student
- DELETE student

## How to Run
1. Install Node.js
2. Run: node server.js
3. Test using Postman

## Endpoints
GET /students  
GET /students/:id  
POST /students  
PUT /students/:id  
DELETE /students/:id
